Laravel Commands

ddd()

Creating table for migrations command:

php artisan make:migration create_(name)_table

Running the migrations command:

php artisan migrate

Updating tables in migrations command:

php artisan make:migraton create_name_table --table=(name)

Rolling back if there are mistakes in migrations command:

php artisan migrate:rollback

Running database seeding commang:

php artisan db:seed

Clearing out the seeds command:

php artisan migrate:refresh

Rolling seeds again command:

php artisan migrate:refresh --seed

Creating models command:

php artisan make:model (name)

Creating factories command:

php artisan make:factory (name)

Creating controller command: 

php artisan make:controller (name)

Creating components command:

php artisan make:component ComponentName
example:
php artisan make:component Navbar
or pede rin 
php artisan make:component Pages/Navbar

Creating middleware command:

php artisan make:middleware (name)
